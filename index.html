📅 SmartCalendar

클립보드의 텍스트를 Google Calendar에 자동 등록하는 AI 기반 생산성 도구

🚀 주요 기능

📅

스마트 일정 등록

클립보드의 텍스트를 Google Calendar에 자동으로 등록합니다. 복사한 텍스트가 자동으로 일정으로 변환됩니다.

🤖

AI 기반 분석

Google Gemini API를 활용하여 텍스트를 스마트하게 분석하고 적절한 일정 정보를 추출합니다.

📝

자동 요약

일정 내용을 AI가 자동으로 요약하여 깔끔하게 정리된 형태로 캘린더에 등록합니다.

🔄

원클릭 등록

확장 프로그램 아이콘을 클릭하면 클립보드의 텍스트가 자동으로 로드되어 즉시 일정 등록이 가능합니다.

🌐

웹 통합

웹 페이지에서 텍스트를 선택하고 우클릭 메뉴를 통해 바로 일정으로 등록할 수 있습니다.

⚙️

간편 설정

API 키 설정과 기본 옵션을 쉽게 구성할 수 있는 직관적인 설정 페이지를 제공합니다.
