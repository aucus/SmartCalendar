📄 개인정보처리방침

SmartCalendar Chrome 확장 프로그램

1. 개인정보 수집 및 이용 목적

SmartCalendar는 사용자의 일정 관리를 돕기 위해 다음과 같은 정보를 수집하고 이용합니다:

  • 클립보드 텍스트: 사용자가 복사한 텍스트를 일정으로 변환하기 위해 임시로 처리
  • Google Calendar 데이터: 일정 등록 및 조회를 위해 Google Calendar API를 통해 접근
  • 설정 정보: API 키 및 사용자 설정을 로컬에 저장

2. 개인정보 수집 방법

다음과 같은 방법으로 개인정보를 수집합니다:

  • 사용자가 직접 복사한 클립보드 텍스트
  • Google OAuth 인증을 통한 Google Calendar 접근 권한
  • 확장 프로그램 설정 페이지를 통한 사용자 설정

3. 개인정보 보호 및 보안

데이터 저장

  • 모든 개인정보는 사용자의 로컬 브라우저에만 저장됩니다
  • 외부 서버로 개인정보를 전송하지 않습니다
  • API 키는 로컬 저장소에 암호화되어 저장됩니다

데이터 처리

  • 클립보드 텍스트는 Google Gemini API를 통해 처리되며, 처리 후 즉시 삭제됩니다
  • 일정 정보는 Google Calendar API를 통해 직접 처리됩니다
  • 중간 서버를 거치지 않고 직접 API 호출을 수행합니다

4. 제3자 서비스

SmartCalendar는 다음 제3자 서비스를 이용합니다:

  • Google Gemini API: 텍스트 분석 및 일정 정보 추출
  • Google Calendar API: 일정 등록 및 관리
  • Google OAuth: 사용자 인증 및 권한 관리

이러한 서비스들의 개인정보처리방침은 각 서비스 제공업체의 정책을 따릅니다.

5. 개인정보 보유 및 이용기간

  • 클립보드 텍스트: 일정 등록 완료 후 즉시 삭제
  • 설정 정보: 확장 프로그램 삭제 시까지 보유
  • Google Calendar 데이터: Google의 데이터 보관 정책에 따름

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 개인정보 수집 및 이용에 대한 동의 철회
  • 저장된 설정 정보 삭제
  • Google Calendar 접근 권한 해제
  • 확장 프로그램 완전 삭제

7. 개인정보처리방침 변경

이 개인정보처리방침은 법령 및 방침에 따라 변경될 수 있으며, 변경 시 GitHub 저장소를 통해 공지합니다.
