SmartCalendar Chrome Extension

1. Purpose of Personal Information Collection and Use

SmartCalendar collects and uses the following information to help users manage their schedules:

  • Clipboard Text: Temporarily processes text copied by users to convert it into calendar events
  • Google Calendar Data: Accesses through Google Calendar API for event registration and retrieval
  • Settings Information: Stores API keys and user settings locally

2. Methods of Personal Information Collection

Personal information is collected through the following methods:

  • Clipboard text directly copied by users
  • Google Calendar access permissions through Google OAuth authentication
  • User settings through the extension settings page

3. Personal Information Protection and Security

Data Storage

  • All personal information is stored only on the user's local browser
  • No personal information is transmitted to external servers
  • API keys are stored encrypted in local storage

Data Processing

  • Clipboard text is processed through Google Gemini API and deleted immediately after processing
  • Calendar information is processed directly through Google Calendar API
  • Direct API calls are performed without going through intermediate servers

4. Third-Party Services

SmartCalendar uses the following third-party services:

  • Google Gemini API: Text analysis and schedule information extraction
  • Google Calendar API: Event registration and management
  • Google OAuth: User authentication and permission management

The privacy policies of these services follow the policies of each service provider.

5. Personal Information Retention and Usage Period

  • Clipboard text: Deleted immediately after event registration completion
  • Settings information: Retained until extension deletion
  • Google Calendar data: Follows Google's data retention policy

6. User Rights

Users have the following rights:

  • Withdraw consent for personal information collection and use
  • Delete stored settings information
  • Revoke Google Calendar access permissions
  • Completely delete the extension

7. Changes to Privacy Policy

This privacy policy may be changed according to laws and policies, and will be notified through the GitHub repository when changes occur.

8. Contact Information

If you have any questions regarding personal information processing, please contact us through the following methods:
