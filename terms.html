📋 서비스 약관

SmartCalendar Chrome 확장 프로그램

1. 서비스 개요

SmartCalendar는 클립보드의 텍스트를 Google Calendar에 자동으로 등록하는 Chrome 확장 프로그램입니다. 본 약관은 SmartCalendar 서비스 이용과 관련된 조건을 규정합니다.

2. 서비스 이용 조건

2.1 이용 자격

  • Chrome 브라우저를 사용하는 사용자
  • Google 계정을 보유한 사용자
  • 본 약관에 동의하는 사용자

2.2 필수 요구사항

  • Google Gemini API 키
  • Google Calendar 접근 권한
  • 안정적인 인터넷 연결

3. 서비스 기능

  • 클립보드 텍스트를 Google Calendar 일정으로 변환
  • AI 기반 텍스트 분석 및 일정 정보 추출
  • Google Calendar API를 통한 일정 등록
  • 웹 페이지 컨텍스트 메뉴를 통한 일정 등록
  • 확장 프로그램 설정 관리

4. 사용자의 의무

  • 본 약관을 준수할 의무
  • 타인의 권리를 침해하지 않을 의무
  • 서비스를 불법적인 목적으로 사용하지 않을 의무
  • API 키 등 민감한 정보를 안전하게 관리할 의무
  • 서비스 이용 중 발생하는 문제에 대한 책임

5. 제한사항 및 면책조항

5.1 서비스 제한

  • Google API의 사용 제한 및 할당량에 따른 제한
  • 인터넷 연결 상태에 따른 서비스 이용 제한
  • 브라우저 호환성 문제로 인한 제한

5.2 면책조항

  • 서비스 이용으로 인한 데이터 손실에 대한 책임 면제
  • 제3자 서비스(Google API)의 장애로 인한 서비스 중단 면제
  • 사용자의 부주의로 인한 보안 문제 면제
  • 서비스의 정확성이나 완전성에 대한 보장 면제

6. 지적재산권

  • SmartCalendar는 MIT License 하에 배포됩니다
  • 사용자는 소스 코드를 자유롭게 수정하고 배포할 수 있습니다
  • Google API 및 관련 서비스는 Google의 지적재산권입니다
  • 사용자가 생성한 콘텐츠의 지적재산권은 사용자에게 있습니다

7. 개인정보 보호

개인정보 수집 및 이용에 대해서는 별도의 개인정보처리방침을 따릅니다.

  • 개발자는 서비스의 기능을 변경하거나 중단할 수 있습니다
  • 중요한 변경사항은 GitHub를 통해 사전 공지합니다
  • 사용자는 언제든지 서비스 이용을 중단할 수 있습니다

9. 분쟁 해결

  • 서비스 이용과 관련된 분쟁은 GitHub Issues를 통해 해결합니다
  • 법적 분쟁이 발생할 경우 대한민국 법률에 따라 해결합니다
  • 분쟁 해결을 위한 소통은 건설적이고 존중하는 태도로 진행합니다
